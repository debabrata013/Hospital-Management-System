## üö® Critical Migration Points

### **1. Field Name Changes**
- `dateOfBirth` ‚Üí `date_of_birth`
- `contactNumber` ‚Üí `contact_number`
- `_id` ‚Üí `id` (auto-increment)

### **2. Data Type Changes**
- MongoDB ObjectId ‚Üí MySQL INT (auto-increment)
- Date strings ‚Üí MySQL DATE/DATETIME
- Nested objects ‚Üí JSON columns or separate tables

### **3. Relationship Changes**
- MongoDB references ‚Üí MySQL foreign keys
- Embedded documents ‚Üí Separate tables with relationships

### **4. Query Changes**
- MongoDB aggregation ‚Üí MySQL JOINs
- `$lookup` ‚Üí `LEFT JOIN`
- `$match` ‚Üí `WHERE`
